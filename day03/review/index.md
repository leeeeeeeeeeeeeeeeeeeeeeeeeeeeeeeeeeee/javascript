# 변수
-데이터를 저장하는 공간
- const/let/--var()
-메모리에 데이터를 저장하기 위해서 (RAM)
-컴퓨터는 기억과 명령을 따로하기 때문에 변수 사용함!

# 데이터 타입
-기본 타입: string, Number, Boolean, Null, Underfined
-참조 타입: Array[], object{key:value}
-타입 태스팅(바꾸기)

# 연산자
-산술 연상자,논리연산자,비교연산자,대입연산자,삼항연산자. . .
-논리연산자에서 단축 평가 ( short-circuit)을 사용해서 코드 짧게 가능!
